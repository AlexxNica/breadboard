Detailed Description

The GraphFactory is a base class that can be used to load Graphs.

This is a utility class that simplifies the process of loading graphs. It handles loading the file into memory and caching previously loaded files. If the same file is requested multiple times, the cached graph is returned. To use this class ParseData must be overridden to translate the data into nodes and edges, and to parse default values.
