#include <type.h>

Detailed Description

Metadata about types that are used as input and output edge parameters on nodes.

Type specifies the size and alignment of a given type, as well as how to construct them using placement new in a buffer. By default they use DefaultPlacementNew, which simply constucts an object at the given address. If the default constructor can not be used, custom a PlacementNewFunc may be supplied.

Note
This class should not be instantiated directly. Instead, Types should be registered through the TypeRegistry.
